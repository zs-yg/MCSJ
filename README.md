# MCSJ 服务器管理工具

MCSJ 是一个开源的 Minecraft 服务器管理工具，旨在为用户提供极致便捷的服务器搭建与管理体验。

## 主要功能

- 一键下载 server.jar
- 一键生成启动脚本
- 一键启动服务器
- 一键下载 Java 环境
- 一些有趣实用的辅助功能

> 更多辅助功能请下载体验。

## 开发环境

- Windows 10
- Visual Studio Code
- .net 8.0

## 运行环境

- Windows 10
- .net 8.0 runtime

## 运行方式

1. 下载并解压 MCSJ 压缩包。
2. 双击运行 MCSJ.exe 启动程序。
3. 或者命令行输入MCSJ.exe 启动程序。可以查看是否需要安装运行时
